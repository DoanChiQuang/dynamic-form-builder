Form Schema Builder form-builder.js

Dùng FormSchemaBuilder để tạo Schema JSON bằng code thay vì viết JSON thủ công. Kết quả có thể dùng trực tiếp với FormRenderer hoặc lưu lên server.

1. Khai báo schemanew FormSchema({ title, key_name })
2. Thêm Boxschema.addBox('Tiêu đề box')
3. Thêm Rowbox.addRow(Input({…}), Select({…}))
4. Xuất JSONschema.toJSON() / schema.toString()
